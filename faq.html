常见问题Questions, answered

Not covered here? Write to contact@scrollery.app — during the beta we read every letter.

Scrollery 是云服务吗?会上传我的照片吗?Is Scrollery a cloud service? Does it upload my photos?

不是,也不会。Scrollery 是一款本地优先的桌面应用:照片、索引、缩略图和 AI 模型推理全部在你的设备上,没有账号体系,断网照常工作。详见隐私承诺。

No and no. Scrollery is a local-first desktop app: your files, the index, thumbnails and AI inference all live on your device. There is no account system, and it works fully offline. See the privacy promise.

现在能下载吗?怎么加入内测?Can I download it now? How do I join the beta?

Windows 版内测进行中,暂无公开下载。发邮件到 contact@scrollery.app,附上你的图库规模(大约多少张)与机器配置,我们按批次发放名额。内测免费。

The Windows build is in private beta; there is no public download yet. Email contact@scrollery.app with your rough library size and machine specs, and seats go out in batches. The beta is free.

支持哪些文件格式?Which file formats are supported?

免费版内置大众常见格式:图像全家(jpg/png/webp/gif/bmp/tif,以及走 Windows GPU 解码的 HEIC/HEIF/AVIF)、常见视频(mp4/mov/mkv/webm 等十种)、常见音频(mp3/flac/wav/aac/m4a)与常见文档(pdf/svg/epub/txt/md),并识别配对 Live Photo。相机 RAW 与更多专业格式(如 PSD)属高级能力,由商店里的签名插件按需提供。完整清单见格式一节。

The free edition covers the everyday formats: the full image family (jpg/png/webp/gif/bmp/tif, plus HEIC/HEIF/AVIF via Windows GPU decode), ten common video formats (mp4/mov/mkv/webm…), common audio (mp3/flac/wav/aac/m4a) and common documents (pdf/svg/epub/txt/md), with Live Photo pairing. Camera RAW and further professional formats such as PSD are premium capabilities, delivered as signed plugins from the store. Full list in the formats section.

它会改动我的文件夹结构吗?Will it touch my folder structure?

不会。Scrollery 沿用你现有的文件夹,原地建立索引:不搬移、不改名、不导入私有库。应用内删除默认进回收站,可反悔。哪天你不再使用,文件仍在原处。

No. Scrollery indexes the folders you already have, in place — no moving, no renaming, no importing into a proprietary vault. In-app deletion goes to a recycle bin first. If you ever stop using it, your files are exactly where they always were.

AI 搜索和人脸识别在哪里运行?Where do AI search and face recognition run?

全部在你的机器上:语义搜索使用 Chinese-CLIP,人脸检测与聚类使用商用许可安全的开源模型,推理跑在独立的本机子进程里(ONNX Runtime / DirectML)。图像内容不会离开你的设备。

Entirely on your machine: semantic search uses Chinese-CLIP, and face detection/clustering uses openly licensed, commercially safe models, all running in a local subprocess (ONNX Runtime / DirectML). Image content never leaves your device.

系统要求是什么?What are the system requirements?

内测阶段面向 64 位 Windows 10 / 11。HEIC/AVIF 解码与 AI 加速会利用系统自带的 GPU 能力(WIC / DirectML),无独立显卡也能运行。macOS 在路线图上。

The beta targets 64-bit Windows 10 / 11. HEIC/AVIF decoding and AI acceleration use the OS's GPU facilities (WIC / DirectML); a discrete GPU is not required. macOS is on the roadmap.

收费吗?Does it cost anything?

内测免费。正式版的商业模式尚未最终确定,会在发布时公布;核心代码保持开源(Apache-2.0)。

The beta is free. Pricing for the final release hasn't been settled and will be announced at launch; the core stays open source under Apache-2.0.

开源吗?我能参与吗?Is it open source? Can I contribute?

核心代码以 Apache-2.0 公开在 GitHub,欢迎围观、提 issue 与反馈;贡献代码需签署 CLA(流程见仓库 CONTRIBUTING)。「Scrollery」名称与标识为商标,授权范围另见 TRADEMARK.md;部分专业格式插件为闭源发行。

The core is public on GitHub under Apache-2.0 — issues and feedback welcome; code contributions require a CLA (see CONTRIBUTING in the repo). The “Scrollery” name and mark are trademarks (see TRADEMARK.md), and some professional-format plugins ship closed-source.

移动硬盘和 NAS 能用吗?Do external drives and NAS work?

能。移动硬盘拔出后,相关条目灰显待命,重新接上即自动恢复;库与缓存也可以放在 WebDAV 网络存储上(例如你自己的 NAS)——数据依然只流向你指定的服务器。

Yes. When a drive is unplugged its items grey out and wait, recovering automatically on reconnect. The library and caches can also live on WebDAV storage (your own NAS, for example) — data still flows only to servers you designate.
