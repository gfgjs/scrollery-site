本

三个立身之本Three things we won't trade away

快,是工程;私,是立场;文件是你的,是承诺。

Speed is engineering. Privacy is a stance. Your files staying yours is a promise.

快

生来为海量Fast — built for big

布局在 Rust 后端算好,前端只画看得见的那几行。数十万张的图库,滚起来依然像一张长卷。

Layout is computed in Rust; the UI paints only the rows you can see. Hundreds of thousands of items still scroll like one long sheet of paper.

  • 两阶段扫描 · 秒级首屏
  • Rust Justified Layout + 行级虚拟化
  • 大库分段虚拟化 + 自研逻辑滚动条
  • WIC GPU 解码(Windows)
私

完全离线Private — fully offline

语义搜索与人脸识别都在你的机器上推理。没有账号,没有云端,断网照常工作。

Semantic search and face recognition run on your machine. No account, no cloud; it works with the network cable unplugged.

  • Chinese-CLIP 本机推理(ONNX / DirectML)
  • 人脸模型商用安全(MIT / Apache-2.0)
  • 无账号 · 无上传 · 离线可用
真

文件始终是你的Yours — files stay files

沿用你已有的文件夹,不搬移、不改名、不锁进私有库。哪天你不用 Scrollery 了,一切原样还在。

It works with the folders you already have — no moving, no renaming, no proprietary vault. Quit Scrollery any day; everything is exactly where it was.

  • Picasa 式文件夹树哲学
  • 原地管理 · 回收站软删除可反悔
  • 核心开源 · Apache-2.0

械

卷底的机械Under the hood

核心代码公开在 GitHub,构建可查验;性能不靠形容词,靠架构。

The core source is public on GitHub with inspectable builds. Performance comes from architecture, not adjectives.

引擎Engine

  • Rust + Tauri 2 — 桌面壳与系统集成desktop shell & OS integration
  • SQLite(WAL)— 索引与元数据,全部本地index & metadata, all local
  • rayon — CPU 并行流水线CPU-parallel pipelines
  • ONNX Runtime + DirectML — AI 独立子进程AI in its own subprocess
  • WIC — GPU 图像解码(Windows)GPU image decode (Windows)

界面Interface

  • Vue 3 + TypeScript — strict mode
  • 原生 CSS 变量主题系统Native CSS-variable theming — 6 themes
  • 行级虚拟化 + Canvas 网格Row virtualization + canvas grid — DOM 恒定constant DOM
  • 后端布局缓存,前端只取视口Backend layout cache; UI pulls the viewport

性能口径:在数十万张的真实图库上日常实测开发,架构面向十万~百万张设计。数字为开发机内部实测,非正式基准。 Honest numbers: developed and measured daily against a real library of hundreds of thousands of items, architected for 100k–1M. Internal dev-machine measurements, not formal benchmarks.
