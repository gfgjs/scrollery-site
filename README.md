# scrollery-site

[Scrollery](https://scrollery.app) 官网静态源码。The static source of the Scrollery landing page.

- **形态**:单文件 `index.html`,零依赖、零构建、零跟踪脚本;双主题(跟随系统 + 手动切换)、中英双语(跟随浏览器 + 手动切换)。
- **部署**:Cloudflare Pages(Git 集成,无构建步骤,输出目录 `/`),自定义域 `scrollery.app`。push 到 `main` 即自动发布。
- **文案纪律**:对外宣称须与已实测能力对齐——性能表述用「数十万张」量级;未落地能力只出现在「路线图」措辞中,不作为现有功能宣称。
